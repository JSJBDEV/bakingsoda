# A Mod About Baking Soda
The most famous use for baking soda for most people is that of the "Baking Soda Volcano" experiment where mixing it with vinegar causes a kind of eruption.

This mod is a submission to ModFest 26 "Animation, Affection, Academia!" of which I'm sure you can see which category it fits in.

![Baking Soda Volcano](https://cdn.modrinth.com/data/cached_images/eef5351b48bcf21e0ea36ece24bf677a8512cb92.png)

The main feature of this mod is of course, the afformention volcano, which you craft with stone, iron and vinegar (which is obtained by smelting any potion).

![Erupting volcano](https://cdn.modrinth.com/data/cached_images/6fa24b03b4d25da84fd8272f74032400a92288d6.png)

Dropping baking soda (found by mining Trona ore (looks like smooth quartz) and then crafting it) into the volcano causes it to erupt.

![Soda Bread](https://cdn.modrinth.com/data/cached_images/070daddbdd8d62d465a725889e06b3cd2a5a62a9.png)

Combining a baking soda with 3 wheat gives you 3 bread, this might be soda bread in a future update

![Fire Extinguisher](https://i.ibb.co/Z6KVqHZh/ezgif-3d2d134cb4c67897.gif)

A fire extinguisher can be crafted, a baking soda from the offhand is used to extinguish all fires within a 5 block radius
